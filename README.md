# AIAnClub

**常用深度学习、计算机视觉算法合集**

## 代码结构
```shell
|---- AIAnClub
    |---- ComputerVision # 传统计算机视觉算法
    |---- DataBase       # 各种数据集存放位置
    |---- OCR            # 文字字符识别算法
    |---- ObjectDetection # 目标检测算法
    |---- PoseEstimation  # 姿态识别算法
```

## 使用

### 编译工具
**建议使用PyCharm**

### 方法
使用pyCharm进入项目的二级目录打开子项目即可使用调试
